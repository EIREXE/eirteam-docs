Welcome to the official documentation of EIRTeam's modules for `Godot Engine <https://godotengine.org>`_,
these are a set of open source modules used in various production games such as Project Heartbeat

The table of contents in the sidebar should let you easily access the documentation
for your topic of interest. You can also use the search function in the top-left corner.

Repositories
------------

You can find the repositories for each module here:

- `EIRTeam.Steamworks <https://github.com/EIRTeam/EIRTeam.Steamworks>`_
- `EIRTeam.InputGlyphs <https://github.com/EIRTeam/EIRTeam.InputGlyphs>`_
- `EIRTeam.FFmpeg <https://github.com/EIRTeam/EIRTeam.FFmpeg>`_

Supporting development
----------------------

If you are interested in donating to EIRTeam as a thank you for these modules please check out `the patreon <https://www.patreon.com/EIRTeam>`_.

It helps me pay the bills :P
